Loja Virtual

Sobre o Projeto

Um site de e-commerce simples (loja virtual) onde os usuários podem navegar por uma lista de produtos e adicioná-los a um carrinho de compras. O foco é criar a estrutura inicial de como um cliente interage com uma loja online.

Respostas do Miniteste

1) Quais são as principais funcionalidades...

  • Visualização de Produtos: O usuário pode ver uma página principal com todos os produtos disponíveis (foto, nome, preço).
  • Detalhes do Produto: (Opcional) Clicar em um produto levaria a uma página com mais detalhes.
  • Carrinho de Compras: O usuário pode adicionar produtos a um carrinho.
  • Finalização de Compra (Simulada): O usuário pode ir para uma página de "checkout" para ver o total e (simuladamente) "finalizar" a compra, talvez preenchendo um formulário de nome e endereço.

2) Quais características de qualidade...

  • Usabilidade: O sistema deve ser extremamente fácil e intuitivo. O usuário deve conseguir encontrar um produto e adicioná-lo ao carrinho sem confusão.
  • Desempenho: As páginas, especialmente a de listagem de produtos, devem carregar rapidamente para não frustrar o cliente.
  • Segurança: Mesmo sendo um projeto inicial, a segurança é vital. Em uma loja real, isso protegeria os dados de login e pagamento do cliente.
